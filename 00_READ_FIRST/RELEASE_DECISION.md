# 公開判定

**版**：v1.0  
**発行日**：2026-07-03

## 判定

正式公開版として配布可能。

## 根拠

- 本文、書式、PDF、HTML、Google実装版を同梱済み。
- 禁止方式を明確化済み。
- PDF、HTML、ZIPの基本的な生成・破損検査を実施済み。
- SHA256SUMSを生成済み。

## 利用開始前に各利用者が行うこと

- 自校名・PTA名・金額・担当者等の空欄を埋める。
- Google実装版は自分のGoogleアカウントでコピー・権限設定・テスト送信を行う。
- 学校・教育委員会に提出する前に、該当自治体の規則・規程と整合確認する。
