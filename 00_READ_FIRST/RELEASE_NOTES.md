# リリースノート

## v1.0

**発行日**：2026-07-03  
**発行**：PTA適正化推進委員会

v1.0は、PTA適正運営スターターキットの正式公開版である。

### 同梱内容

- マニュアル本文充実版
- 標準書式14種
- PTA会則モデル
- 移行ロードマップ
- 学校・教育委員会向け説明書
- 反論対応集
- 情報公開・監査確認項目
- 詳細チェックリスト
- 一次資料リンク集
- PDF印刷版
- HTMLサイト掲載版
- Googleフォーム・スプレッドシート・Apps Script実装パッケージ

### 基本方針

- PTAは学校ではない
- PTA加入は契約である
- 入会しない者は何も提出しなくてよい
- 非加入届・踏絵方式・非会員児童一覧は作らない
- 学校名簿、学校回収、学校徴収、教職員事務依存から分離する
- 学校管理職の当該学校PTA役員就任を制度設計上禁止する

### v1.0で意図的に含めないもの

- 非加入届フォーム
- 加入・非加入二択フォーム
- 未提出者督促機能
- 学校名簿インポート・照合機能
- 非会員児童一覧
- 役員免除申請
- 家庭事情申告

### 今後の更新候補

- 実運用でのApps Script不具合修正
- PDF版の体裁改善
- 各自治体通知・回答例の追加
- 法令資料リンクの更新
- Word版・Googleドキュメント版の作成
