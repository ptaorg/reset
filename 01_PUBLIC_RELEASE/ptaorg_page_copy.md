# PTA適正運営スターターキット v1.0

PTAを、学校依存型の慣行から、任意加入・個人情報保護・会計分離・学校との公私分離に基づく適正運営へ移行するための実務パッケージです。

本キットは、単なる入会申込書のテンプレート集ではありません。自動加入、みなし加入、非加入届、踏絵方式、学校名簿利用、学校回収、学校徴収、教職員事務依存、学校管理職のPTA役員就任を前提にしない、PTA運営全体の再設計パッケージです。

## 含まれるもの

- マニュアル本文
- 標準書式14種
- PTA会則モデル
- 移行ロードマップ
- 学校・教育委員会向け説明書
- 反論対応集
- 情報公開・監査確認項目
- 詳細チェックリスト
- 一次資料リンク集
- PDF印刷版
- HTML掲載版
- Googleフォーム・スプレッドシート・Apps Script実装版

## 作らないもの

- 非加入届
- 加入・非加入二択フォーム
- 未提出者督促
- 学校名簿照合
- 非会員児童一覧
- 役員免除申請
- 家庭事情申告
