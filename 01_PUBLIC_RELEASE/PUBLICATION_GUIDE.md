# PUBLICATION GUIDE

## 1. 公開方法の推奨

公開は次の3段階に分けてください。

### 第1段階: PDF配布

学校・教育委員会・PTA役員に見せる場合は、まずPDF版を使います。

- 本文だけ: `02_PDF_PRINT/pta-starter-kit-v0.2.2-rich-manual-print.pdf`
- 全部入り: `02_PDF_PRINT/pta-starter-kit-v0.2.2-rich-full-print.pdf`
- 書式だけ: `02_PDF_PRINT/pta-starter-kit-v0.2.2-rich-templates-print.pdf`

### 第2段階: HTML公開

サイト掲載は、`03_HTML_SITE/pta-starter-kit-v0.2.2-rich-site.zip` を展開して使います。

入口は `index.html` です。

### 第3段階: 実務運用版の提供

実際にPTAが導入する場合は、`05_GOOGLE_IMPLEMENTATION` を使います。最初は必ずテスト用Googleアカウント・テスト用スプレッドシートで確認してください。

## 2. 公開時の注意

- これは法律相談ではなく、標準運用モデルであることを明記する
- 自治体規則・学校管理規則・私費会計規程の確認が必要であることを明記する
- 公開日と版番号を明記する
- 旧版と混同しないようにする
- Google実装版は本番導入前にテストするように書く
