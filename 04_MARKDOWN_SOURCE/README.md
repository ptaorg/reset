# PTA適正運営スターターキット v0.2.2-rich

## 位置付け

v0.2.2-rich は、v0.2.1 の「薄い要約版」を実務で使える水準まで増補した版です。

v0.2.1 は、本文が短く、学校・教育委員会・PTA役員に説明するための材料、反論処理、規約改正、移行工程が不足していました。本版では、次を追加しています。

- 22章すべてを実務説明つきに増補
- 各章に「不適正運用の典型例」「法令・制度上の整理」「標準方式」「学校への説明ポイント」「確認項目」を追加
- PTA会則モデルを追加
- 学校・教育委員会向け説明書を追加
- 導入ロードマップを追加
- 反論対応集を追加
- 情報公開・監査確認項目を追加
- 一次資料リンク集を追加
- HTML化前のMarkdown実務版として整理

## 使い方

1. `01_マニュアル本文_充実版.md` を読む。
2. `06_移行ロードマップ.md` で自校の段階を確認する。
3. `05_PTA会則モデル.md` をもとに規約改正案を作る。
4. `07_学校教育委員会向け説明書.md` を学校・教委への説明に使う。
5. `08_反論対応集.md` を役員会・総会・保護者説明で使う。
6. 必要な書式は `templates/` から取り出す。

## 本版で特に強化した点

- 「非加入届を取らない」だけでなく、なぜ非加入届が非会員情報の取得になるのかを説明。
- 「学校名簿を使わない」だけでなく、公立学校側とPTA側の個人情報規律を二層で説明。
- 「学校回収しない」だけでなく、担任回収・児童経由・連絡帳経由が学校提出物の外観を作る点を説明。
- 「会費分離」だけでなく、委任が非会員に会費債務や同意を発生させない点を説明。
- 「管理職役員禁止」について、現行法上一律違法断定ではなく制度設計上の禁止として整理。
- 「学校施設利用」について、当然利用ではなく許可・条件・費用・責任の問題として整理。
