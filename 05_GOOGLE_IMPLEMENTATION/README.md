# PTA適正運営スターターキット v0.3.1 実装パッケージ

このパッケージは、v0.3設計版を実際のGoogleフォーム・スプレッドシート運用に近づけた実装版です。

## v0.3.1で強化した点

- Apps Scriptに管理メニュー `PTA適正運営 v0.3.1` を追加
- 必須シート・ヘッダーを自動作成・修復する `ptaInstallOrRepair()` を追加
- フォーム作成・トリガー設定を `ptaCreateForms()` に整理
- 既存フォームIDがあれば再利用する方式に変更
- フォームURL一覧シートを自動更新
- 受付番号発番に `LockService` を使用
- 行追加をヘッダー名基準に変更し、列ずれに強くした
- 入会時に会員台帳と会費管理へ同時登録
- 退会時に退会管理と削除予定へ登録
- 会員情報更新受付シートを追加
- 自己点検シートと `ptaRunSelfCheck()` を追加
- 削除予定通知トリガーを追加
- 禁止機能が混入していないか確認するチェックを追加

## 作るもの

- 入会申込フォーム
- 退会通知フォーム
- 会員情報更新フォーム
- 役員立候補・承諾フォーム
- 会員台帳
- 会費管理
- 退会管理
- 更新受付
- 役員承諾
- 権限管理
- 削除予定
- 操作ログ
- 自己点検

## 作らないもの

- 非加入届フォーム
- 加入・非加入二択フォーム
- 未提出者督促
- 学校名簿インポート
- 学校名簿照合
- 非会員児童一覧
- 未加入者一覧
- 学級別加入率集計
- 役員免除申請
- 家庭事情申告

## 最初に読むファイル

1. `INSTALL_GUIDE.md`
2. `OPERATIONS_MANUAL.md`
3. `TEST_PLAN.md`
4. `SECURITY_CHECKLIST.md`
5. `scripts/Code.gs`

## 注意

このパッケージはGoogle側へ自動デプロイするものではありません。Googleスプレッドシートにインポートし、Apps Scriptエディタに `Code.gs` を貼り付け、ユーザー自身が権限承認を行う必要があります。
