# v0.3.1 導入手順

## 1　スプレッドシートを準備する

1. `xlsx/pta-v0.3-sheet-template.xlsx` をGoogleドライブへアップロードする。
2. 右クリックして「Googleスプレッドシートで開く」。
3. ファイル名を `PTA会員台帳 v0.3.1` 等に変更する。
4. 学校アカウントではなく、PTA管理のGoogleアカウントで管理する。

## 2　Apps Scriptを貼り付ける

1. Googleスプレッドシートで「拡張機能」→「Apps Script」を開く。
2. 既存コードを削除する。
3. `scripts/Code.gs` の全文を貼り付ける。
4. プロジェクト設定でタイムゾーンを `Asia/Tokyo` にする。
5. 必要に応じて `scripts/appsscript.json` の内容をマニフェストへ反映する。

## 3　初期シートを作成・修復する

Apps Scriptで `ptaInstallOrRepair()` を実行する。

初回実行時はGoogleの権限承認が必要になる。

実行後、次のシートが存在することを確認する。

- コード設定
- フォームURL
- 会員台帳
- 会費管理
- 退会管理
- 更新受付
- 役員承諾
- 権限管理
- 削除予定
- ログ
- 自己点検

## 4　コード設定を編集する

`コード設定` シートで次を設定する。

| キー | 設定内容 |
|---|---|
| PTA_NAME | PTA名称 |
| ADMIN_EMAIL | 管理者通知先メール |
| PRIVACY_POLICY_URL | プライバシーポリシーURL |
| FEE_AMOUNT | 年会費額 |
| FEE_YEAR | 会費年度 |
| DELETION_AFTER_DAYS | 退会後の削除予定日数 |

## 5　フォームを作成する

`ptaCreateForms()` を実行する。

作成されるフォームは次の4つだけ。

- 入会申込フォーム
- 退会通知フォーム
- 会員情報更新フォーム
- 役員立候補・承諾フォーム

非加入届フォーム、加入・非加入二択フォーム、役員免除申請フォームは作成しない。

## 6　フォームURLを確認する

`フォームURL` シートに、編集URLと回答URLが出力される。

保護者に案内するのは回答URLのみ。編集URLは管理者だけが保持する。

## 7　削除予定通知を設定する

必要に応じて `ptaInstallDeletionReminder()` を実行する。

削除予定日が30日以内の記録がある場合、管理者へ通知する。

## 8　自己点検を実行する

`ptaRunSelfCheck()` を実行する。

`自己点検` シートでNGが出ていないか確認する。
