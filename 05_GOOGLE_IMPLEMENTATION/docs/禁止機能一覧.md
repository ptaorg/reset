# 禁止機能一覧

v0.3.1では、次の機能を作らない。

## 作らないフォーム

- 非加入届フォーム
- 加入・非加入二択フォーム
- 未提出者確認フォーム
- 役員免除申請フォーム
- 家庭事情申告フォーム
- 診断書提出フォーム
- 勤務証明提出フォーム

## 作らないシート

- 非会員児童一覧
- 非会員家庭一覧
- 未加入者一覧
- 未提出者一覧
- 学校名簿
- 学校名簿照合
- 学級別加入率
- 役員免除者一覧
- 家庭事情一覧

## 作らない処理

- 学校名簿インポート
- 学校名簿照合
- 未提出者督促
- 非加入家庭抽出
- 非会員児童抽出
- 加入率ランキング
- 学級別未提出者集計

## 理由

本キットは、入会した会員だけを管理するためのものです。入会しない者、未提出者、非会員児童を把握するための機能は、任意加入・個人情報保護・児童への不利益取扱い防止と両立しません。
