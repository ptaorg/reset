# v0.3.1 運用マニュアル

## 入会受付

1. 入会希望者だけが入会申込フォームを送信する。
2. 入会しない者には何も提出させない。
3. 回答があると `会員台帳` に登録される。
4. 同時に `会費管理` に会費確認行が作成される。
5. 管理者へ通知が送られる。

## 退会受付

1. 会員本人が退会通知フォームを送信する。
2. 退会は許可制ではない。
3. 退会理由・家庭事情は取得しない。
4. 回答は `退会管理` に記録される。
5. `削除予定` に保存期限管理用の行が追加される。

## 会員情報更新

1. 会員本人が会員情報更新フォームを送信する。
2. 回答は `更新受付` に記録される。
3. 管理者が本人確認のうえ、会員台帳を手動更新する。
4. 学校名簿との照合は行わない。

## 役員承諾

1. 本人が役員立候補・承諾フォームを送信する。
2. 回答は `役員承諾` に記録される。
3. 役員就任は本人承諾制である。
4. くじ引き、免除申請、家庭事情申告の代替にしてはならない。

## 会費管理

会費は、入会した会員に対してPTAが直接案内する。

学校徴収金、学校口座、学校事務職員、学校集金袋を使用しない。

## 権限管理

閲覧・編集権限は必要最小限にする。

退任者の権限は即時削除する。

## 削除予定管理

退会者・卒業者等の情報は、保存期間経過後に削除又は匿名化する。

削除予定通知は、削除を自動実行するものではない。管理者に確認を促す機能である。
